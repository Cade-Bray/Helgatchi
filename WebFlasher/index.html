Rules

regex help Regex patterns (the "matches (regex)" mode)
Case-insensitive; must match the whole value.

. any character
* + ? repeat previous (0+, 1+, 0-1)
\d digit   \w letter/digit/_   \s space
[a-z] any in set   [^0-9] any not in set
\. a literal dot (backslash escapes specials)

Examples
BWL\d-\d* → BWL9-089912, BWL5-012241
cam-\d+ → cam-1, cam-42
[a-f0-9]* → any hex string

Tip: "contains / starts with / ends with" already cover plain text — reach for regex only when the value has a pattern like these.
