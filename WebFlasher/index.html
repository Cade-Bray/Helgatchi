Helgatchi Web Flasher

Flash firmware to your ESP32-S3 handheld — no tools to install.

Install firmware

Your browser can't flash over USB. Use desktop Chrome, Edge, or Opera. Serial access was blocked. Reload and allow the port when prompted.

Before you start

  • Use desktop Chrome, Edge, or Opera (Web Serial isn't in Firefox/Safari).
  • Connect the Helgatchi with a data USB-C cable (not charge-only).
  • Pick a version above, click Connect & Install, and select the serial port.
  • First install erases the chip; settings return to defaults on first boot.
